# Open-Source-Projekte für FHIR-Testautomatisierung

Hier ist eine Übersicht der gefundenen Open-Source-Projekte im Zusammenhang mit FHIR-Testautomatisierung:

| Projektname | Beschreibung | Programmiersprache | Lizenz | Repository-URL | Aktivität | Fokus |
|------------|--------------|-------------------|---------|----------------|------------|--------|
| **TestScript Engine** | Open-Source-Engine zum Ausführen von Tests, die als HL7 FHIR TestScript Ressourcen beschrieben sind. Erzeugt Ergebnisse als FHIR TestReport und unterstützt Integration mit anderen Tools. | Ruby | Apache-2.0 | [GitHub](https://github.com/fhir-crucible/testscript-engine) | Letzter Commit Dez 2022 | Ausführung von TestScripts (Interpreter für FHIR TestScript, erzeugt TestReport) |
| **TestScript Generator** | Open-Source-Befehlszeilentool zur Generierung von HL7 FHIR TestScript Ressourcen aus FHIR Implementation Guides (IGs). Ziel ist die automatische Erstellung einer TestSuite in Form von TestScripts basierend auf IG-Definitionen. | Ruby | Apache-2.0 | [GitHub](https://github.com/fhir-crucible/testscript-generator) | Letzter Commit Dez 2022 | Generierung von TestScripts (erstellt TestScript-Suiten aus IGs) |
| **testscript-eval-py** | Python-basierte Engine zum Ausführen von FHIR TestScripts. Bietet einen Pytest-Plugin, der TestScript-Dateien (in YAML) automatisch erkennt und ausführt, um FHIR-Server-Interaktionen zu verifizieren. | Python | MIT | [GitHub](https://github.com/beda-software/testscript-eval-py) | Aktiv (Last Commit Nov 2023) | Ausführung von TestScripts (Pytest-basiertes Ausführen von YAML-TestScript Dateien) |
| **Inferno** | Open-Source-Framework für Konformitätstests von FHIR APIs. Enthält eine Web-Anwendung und CLI zum Ausführen vordefinierter Test Kits (z.B. für US Core, SMART IG) oder eigener Tests mithilfe einer Ruby-basierten DSL. | Ruby | Apache-2.0 | [GitHub/Docs](https://inferno-framework.github.io/docs/) | Aktiv (laufende Weiterentwicklung, 2023) | Konformitätstests für FHIR (Testkits mit Szenarien; eigenständige Test-Apps auf Inferno-Core-Basis) |
| **Crucible** | Suite von FHIR-Testtools zur Überprüfung der Konformität von FHIR-Servern. Unterstützt mehrere FHIR-Versionen (DSTU2, STU3, R4), bietet umfassende Tests der FHIR-Schnittstelle sowie einen Scorecard-Analyzer für FHIR-Daten. | Ruby (Server+UI) | Apache-2.0 | [Website](https://projectcrucible.org) | Letzte Aktualisierung R4 (Stand 2019) | Konformitätstest von FHIR-Implementierungen (inkl. TestScript-Engine und Datenqualität-Scorecards) |
| **Touchstone** | Cloud-basierte FHIR Testing-Plattform (Testing as a Service) der Firma AEGIS. Führt HL7 FHIR TestScripts gemäß dem FHIR Testing Framework aus und stellt tausende vordefinierte Testskripte für FHIR-Server und -Clients bereit. | Nicht öffentlich | Proprietär | [Website](https://touchstone.aegis.net) | Aktiv (laufender Service seit 2015) | Ausführung von TestScripts zur Konformitätsprüfung (FHIR-Tests als Online-Dienst) |
| **FHIR TestCases** | Sammlung von Testfällen zur Prüfung von FHIR-Implementierungen und -Validatoren. Beinhaltet diverse FHIR-Ressourcen und Definitionen, die als Maven-Paket veröffentlicht werden. | Java (Maven-Projekt) | Apache-2.0 | [GitHub](https://github.com/FHIR/fhir-test-cases) | Aktiv (Version 1.7.15 veröffentlicht 2024) | Testfälle für FHIR-Konformität (statische Test-Ressourcen für Validatoren) |
| **NIST FHIR Toolkit** | Open-Source-Testplattform (Codename "Asbestos") von NIST für HL7 FHIR. Beinhaltet einen Java-basierten Test-Engine-Server (mit FHIR TestScript Interpreter) und einen Proxy/Simulator für IHE-Profile wie MHD. | Java (Server) + Vue/JS (Client) | Unentgeltlich (US Govt Open Source) | [GitHub](https://github.com/usnistgov/asbestos) | Aktiv (Release v2.2.1, ca. 2023) | TestScript-Ausführung mit Simulation (prüft FHIR-Implementierungen, v.a. IHE-konforme Workflows) |
| **AU FHIR Test Data** | Sammlung von FHIR-Testdaten und -Ressourcen für das australische Gesundheitssystem. Enthält validierte Beispieldaten für verschiedene FHIR-Profile und Use Cases. | JSON/XML | Apache-2.0 | [GitHub](https://github.com/hl7au/au-fhir-test-data) | Aktiv (regelmäßige Updates) | Testdaten für AU FHIR-Profile |
